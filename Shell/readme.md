### Chapter 3: Basic bash Shell Commands

1. man

&emsp;&emsp; an interface to the on-line reference manuals

    - Name: 显示命令名和相关描述
    - Synopsis: 语法
    - Configuration: 配置信息
    - Description: 命令的一般性描述
    - Options: 命令的选项描述
    - Exit Status: 命令的退出状态指示
    - Return Value: 命令的返回值
    - Errors: 命令的错误信息
    - Enviroment: 描述使用的环境变量
    - Files: 命令用到的文件
    - Versions: 命令的版本信息
    - Conforming To: 命令所遵从的标准
    - Notes: 其他有帮助的资料
    - Bugs: 提交bug的途径
    - Example: 展示命令的用法
    - Authors: 开发人员相关
    - Copyright: 源代码版权情况
    - See Also: 相似命令
